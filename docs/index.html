Zooom3

A macOS utility that adds modifier key + mouse window move and resize, inspired by X11/Linux window managers and the classic Zooom2 app.

Zooom3 is a fork of Easy Move+Resize by Daniel Marcotte, featuring additional enhancements to perfectly recreate the classic Zooom2 experience.

Download for macOS download View GitHub

Powerful Features

Total control over your windows without hunting for edges.

drag_pan

Move Windows

Hold your chosen modifier keys and left-click drag anywhere inside a window to move it instantly.

aspect_ratio

Resize Windows

Right-click drag with modifiers. Direction is based on the cursor's position in a 3x3 grid within the window.

mouse

Hover Mode

No clicks required. Simply hold the modifier keys and windows move or resize as you move the mouse.

settings_input_component

Custom Modifiers

Choose any combination of Cmd, Ctrl, Alt, Shift, or Fn keys to trigger your actions.

tune

Separate Resize

Configure entirely different modifier keys specifically for resizing if you prefer a different feel.

app_registration

Smart Settings

Disable Zooom3 for specific apps and choose which mouse buttons trigger move vs resize.

Download the latest release from GitHub.

2

Unzip the downloaded archive.

3

Drag the Zooom3 app to your Applications folder.

Accessibility Permissions

Zooom3 requires Accessibility access to interact with other application windows.

Go to System Settings > Privacy & Security > Accessibility. Enable Zooom3. If it's already enabled but not working, remove it with the minus (-) button and add it back manually.

security

macOS Security

Zooom3 is currently unsigned. You will see a "Developer cannot be verified" warning when first opening it.

macOS Sequoia (15)+

  1. Attempt to open Zooom3; click "OK" on the warning.
  2. Open System Settings > Privacy & Security.
  3. Scroll to the bottom and click "Open Anyway".

macOS Sonoma (14) & earlier

  1. Right-click (or Control-click) the Zooom3 icon.
  2. Select Open from the shortcut menu.
  3. Click Open in the dialog that appears.

How to Use

Mastering window management in seconds.

⌨️

Shortcuts

Default: Cmd + Ctrl. Click and drag anywhere on a window to move or resize.

✨

Hover Mode

Enable in settings to move/resize just by holding modifiers and moving the mouse—no clicking required.

📍

Menu Bar

Access quick toggles to Disable Zooom3, "Bring Window to Front", or "Reset to Defaults".
